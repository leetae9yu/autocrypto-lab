# autocrypto-lab

Local-first crypto quant research pipeline for portfolio-grade Research Engineer / Quant Dev evidence.

This project intentionally starts as a safe v1 research harness:

- public/free data first;
- BTC, ETH, SOL, XRP canonical universe;
- minimum 1h cadence, no sub-hour scalping;
- no order endpoints or live-trading execution;
- config-only autonomous research loop with auditable ledger;
- outputs are research diagnostics, not investment advice.

Approved planning artifacts live under `.omx/plans/`.
